## [1.0.0] - 2024-06-19

### Added
- Initial release of Lucent Cycle
- 🌙 Cycle tracking with manual period start dates
- 📅 Interactive calendar view with visual predictions
- 💜 Daily symptom logging (cramps, headache, mood, fatigue)
- 🔮 Automatic predictions for ovulation and next period
- 📊 Pattern insights (cycle count, average period length, common symptoms)
- 🧘 Phase-specific relief suggestions (exercise & nourishment)
- 🌓 Dark mode with theme persistence
- 📱 Fully responsive design (mobile, tablet, desktop)
- ⚙️ Customizable settings (cycle length, period length)
- 🔒 100% privacy-first (all data stored locally)
- ♿ Accessibility features (ARIA labels, keyboard navigation)
- 💜 Glass-morphism UI with smooth animations

### Features
- localStorage-based persistence across sessions
- Phase-based cycle recognition (menstrual, follicular, ovulation, luteal)
- Configurable cycle length (20–45 days)
- Configurable period length (2–10 days)
- Browser notification support for reminders
- Daily rotating comfort messages
- Visual color coding for calendar states

### Technical
- Vanilla JavaScript (no dependencies)
- Single-page application
- CSS3 with custom properties and animations
- Semantic HTML5
